## Development

Clone and `npm i`

```bash
$ git clone git@github.com:anqurvanillapy/anqurvanillapy.github.io
$ cd anqurvanillapy.github.io/
$ npm i
```

`npm start` for starting a dev server, `webpack-dev-server` by default

```bash
$ npm start
```
